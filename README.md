# Posture-Reminders
Powershell script that reminds you to fix your posture with Windows notifications.

## Features
A Windows notification will popup every 5 minutes that reminds you to fix your posture.

The script will start automatically in the background every time you log in.

![](https://imgur.com/qZU98As.jpg)

## Installation
Run the "setup.ps1" file. Don't delete the "notification.ps1" file.
